# Clara Landing Page

A responsive Next.js landing page built with Tailwind CSS.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - Reusable React components
- `tailwind.config.js` - Tailwind CSS configuration

## Customization

To match your Figma design:

1. Update colors in `tailwind.config.js` under the `theme.extend.colors` section
2. Adjust typography and spacing in individual components
3. Replace placeholder content with your actual copy
4. Add images and assets as needed

## Build

```bash
npm run build
```

## Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
